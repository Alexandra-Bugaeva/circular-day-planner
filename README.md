# Круговой планер дня

Android-приложение для визуального планирования времени в формате кругового циферблата.

## Стек
- Kotlin
- Jetpack Compose
- Room
- MVVM
- DataStore

## Возможности
- Создание событий
- Круговой циферблат дня
- To-Do список
- Настройки интерфейса
- Локальное хранение данных
